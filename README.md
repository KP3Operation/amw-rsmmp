# AviatMobileWeb

Aviat Mobile Web adalah aplikasi yang akan digunakan untuk kebutuhan pasien dan dokter. Fitur-fitur yang ada di dalam aplikasi ini sebagai berikut: 

1. Patient App : 
    1. Patient Appointment
    2. Encounter History
    3. Lab Result
    4. Prescription History
    5. Vital Sign Chart
2. Dokter App :
    1. Appointment List
    2. Inpatient List
    3. Access to Inpatient Medical Note
    4. Summary Fee.
- Aplikasi ini akan dijalankan pada Web browser mobile Android atau IOS.
- Aplikasi ini dibangun menggunakan framework Laravel, VueJs dan Bootstrap 5.

> Documentation can be found at `docs` folder.
